# 미션 - 자동차 경주(이상준)

## 🛠️ 구현할 기능 목록
1. 레이싱 게임을 시작
- [] 게임 시작 기능

2. 자동차의 이름을 입력
- [] 자동차 이름 입력 받는 기능
- **예외사항**
  - [] 자동차 이름에 공백이 들어 있는 경우
  - [] 이름은 5자 초과일 경우
  - [] 자동차 이름 입력이 쉼표(,)로 끝나는 경우
  - [] 자동차를 이름을 1개도 입력 안한 경우
  - [] 이름이 중복되었을 경우
  - [] 입력된 이름을 통해 자동차 생성

3. 시도 횟수를 입력 받는다.
   [] 시도 횟수를 입력 받는 기능
- **예외사항**
  - [] 입력 값이 숫자가 아닐 경우
  - [] 입력 값이 0이하일 경우

4. 자동차를 이동시키고, 결과를 출력한다.
   - [] 자동차 움직이기
   - [] 이동 결과 출력 기능
   - [] 시도 횟수 만큼 반복 기능
